
<h1 align="center">Obstacle Avoidance Robot</h1>
<p align="justified">
The Obstacle Avoidance Robot is an Arduino Uno R3-based robotic platform capable of operating in both Autonomous and Manual modes. In Autonomous Mode, an HC-SR04 ultrasonic sensor mounted on an SG90 servo motor detects obstacles and enables intelligent path selection, while an L293D Motor Driver Shield controls the DC gear motors. Manual operation is achieved through an HM-10 BLE 4.0 Bluetooth module. The robot also supports an optional cleaning functionality by integrating a front-mounted BO motor with a rotating mop attachment, making it a versatile embedded robotics platform.
</p>
